
# Init commands
> Для якісної ініціалізації `Brain`, викликати по черзі, як указано, при потребі
 
1 - `/init-vector` - Верифікації векторної памʼяті, треба для полегшення подальшої роботи
2 - `/init-brain` - Верифікація інструкцій `Brain`
3 - `/init-agents` - Верифікація агентів `Brain`
4 - `/init-task` - Верифікація поточних завдань проекту. Якщо треба.
